# Blockchain - CLI

# Blockchain command line interpreter

## Installation


### Prerequisities
* OpenSSL packages 

```bash
sudo apt-get install libssl1.0.0
sudo apt-get install libssl-dev
```
* `llist.so` and `llist.h`
```bash
cd llist/
./install.bash
```

### Compilation
Navigate to `cli/`:
```bash
cd cli/
```
```bash
make cli
```

## Use
```bash
./cli
```
to star the cli
